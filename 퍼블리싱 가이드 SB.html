퍼블리싱 가이드 (HTML / CSS)
구분 사용 상황 가이드 설명 복사해서 사용할 코드
제목 페이지 상단 제목 페이지의 주요 제목에는 <h2> 태그를 사용합니다.
디자인은 class로만 제어하며, 태그 변경은 하지 않습니다. 		페이지 제목용 h2 태그 사용
<h2 class="title-lg">페이지 제목</h2>
소제목 섹션 구분 섹션 단위 구분 시 <h3> 태그를 사용합니다.
h2 다음 단계에서만 사용합니다. 		섹션 구분용 h3 태그
<h3 class="title-md">섹션 제목</h3>
본문 설명 텍스트 일반 텍스트는 <p> 태그를 사용합니다.
줄바꿈을 위해 br 태그를 남용하지 않습니다. 		본문 텍스트 p 태그
<p>설명 텍스트를 입력하세요.</p>
리스트 항목 나열 순서 없는 목록은 <ul>과 <li>를 사용합니다.
점 스타일은 CSS로 제어합니다. 		리스트 ul li 사용
<ul class="list-dot"> <li>항목 1</li> <li>항목 2</li> </ul>
버튼 액션 버튼 클릭 액션이 있는 경우에만 <button> 태그를 사용합니다.
링크 이동은 a 태그를 사용합니다. 		버튼 태그 사용
<button class="btn-primary">확인</button>
